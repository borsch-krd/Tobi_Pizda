# Сервис синхронизированных заметок

## Описание
Полнофункциональный сервис для хранения и синхронизации заметок в формате Markdown между различными устройствами. Включает серверную часть на Python с REST API, консольный и GUI клиенты, а также документацию по созданию приложений для Android, Windows и Linux.

## Архитектура
- **Серверная часть** на Python (Flask) с REST API
- **База данных** SQLite для хранения заметок
- **Механизм аутентификации** и синхронизации
- **Клиентские приложения** для Android, Windows и Linux
- **Поддержка формата** Markdown (.md)

## Функциональность
- Создание, редактирование и удаление заметок в формате .md
- Синхронизация между устройствами
- Аутентификация пользователей
- Хранение заметок в формате Markdown
- Рендеринг Markdown в HTML
- Защита данных через авторизацию

```
/workspace/
```

## Установка и запуск
Смотри полную инструкцию в файле [INSTALL.md](INSTALL.md)

### Быстрый старт:
1. Установка зависимостей сервера:
   ```bash
   cd server
   pip install -r requirements.txt
   ```

2. Запуск сервера:
   ```bash
   python app.py
   ```

3. Запуск GUI клиента:
   ```bash
   cd client
   python gui_client.py
   ```

## Тестирование
Проект включает в себя автоматические тесты API, которые успешно проходят все основные сценарии работы:
- Регистрация и аутентификация пользователей
- Создание, чтение, обновление и удаление заметок
- Рендеринг Markdown в HTML
- Защита доступа к заметкам

## Платформы
- **Android**: через Kivy, BeeWare или Chaquopy
- **Windows**: PyQt/PySide с упаковкой через PyInstaller
- **Linux**: PyQt/PySide с созданием .deb пакета
